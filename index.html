DSO Finder

What's this website about?

This website's main purpose is to search for astronomical objects and view images of them from professional sky surveys. There are some other tools available, like field of view calculator.

What can I search for?

You can find basically any astronomical object you want! Numerous galaxies and countless nebulae, Sun, Moon... You can search for objects using:

  • Name (works only for popular objects, like Heart Nebula, Orion Nebula, Horsehead Nebula)
  • Catalog number (like NGC 6888, M27)
  • Coordinates (like 159.684 +48.817)
  • For more examples, go here

When you search for an object, my website will load an image of it. Website loads images from SDSS (better quality than DSS) if available.

What is field of view calculator?

Field of view calculator lets you check how an object will look like through your telescope. You can choose between imaging and observing mode. In imaging mode, you need to input details about your telescope and camera and in observing mode you need to input details about your telescope and eyepiece.
