How to use this website?

On the main screen, type name of object you'd like to find and click "Find object".

Accepted input types

Names:
  • Horsehead Nebula
  • Dumbbell Nebula
  • Andromeda Galaxy
  • Polaris
  • Alpha Cyg
Sun and Moon:
  • Sun (avaiable in white light and H-alpha)
  • Moon
Catalog numbers:
  • M 42
  • NGC 4565
  • UGC 12158
  • PGC 39058
  • Arp 273
  • IC 5070
  • etc.
Coordinates:
  • 159.684 +48.817
  • 0h 42m 44s +41d 16m 9s

I can't find an object!

If you're searching by name, try to find by catalog number.
