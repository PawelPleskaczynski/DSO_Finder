How to use this website?

On the main screen, type name of object you'd like to find and click "Find object".

Accepted input types

Names:
  • Horsehead Nebula
  • Dumbbell Nebula
  • Andromeda Galaxy
  • Polaris
  • Alpha Cyg

Catalog numbers:

  • M 42
  • NGC 4565
  • UGC 12158
  • PGC 39058
  • Arp 273
  • IC 5070
  • etc.

Coordinates:

  • 20 54 05.689 +37 01 17.38
  • 10:12:45.3-45:17:50

I can't find an object!

If you're searching by name, try to find by catalog number or by coordinates (in proper formatting).
